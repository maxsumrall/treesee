# treesee

A pi extension that turns a session into a top-down branch map — so you can see the shape of a conversation at a glance.

Once you've forked a session a few times, individual branches get noisy fast. Treesee gives you the overview: what branches exist, which one is active, and what each was trying to do.

![treesee example](./docs/treesee-example.png)

## Install

```bash
pi install git:github.com/maxsumrall/treesee
```

Then reload pi:

```bash
/reload
```

## Usage

```text
/treesee             # current session
/treesee select      # pick from sessions in the current project
/treesee all         # pick from all projects
/treesee /path/to/session.jsonl
```

Each branch node shows turns, tools used, word count, and duration. Hover for details. Branch labels are generated by your current pi model and cached locally.
